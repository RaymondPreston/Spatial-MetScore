# Spatial-MetScore Analysis Pipeline

This project processes spatial transcriptomics data (Visium) to calculate a rank-based gene signature score (`MetScore`) and perform downstream deconvolution analysis.

The directory structure is organized following the [Python Data Science Project Organization template](https://gist.github.com/ericmjl/27e50331f24db3e8f957d1fe7bbbe510).

## Project Structure

```
/scratch/rprest2/Spatial-MetScore/
|- notebooks/                    # Analysis and visualization notebooks
   |- 01_tissue_metscore.ipynb   # Tissue-level MetScore calculation
   |- 02_metscore_visualization.ipynb # Side-by-side H&E and MetScore spatial plots
   |- 03_deconvolution.ipynb     # Cell type deconvolution with composite references
   |- figures/                   # Output figures generated by notebooks
|- spatial_metscore/             # Local Python package
   |- spatial_metscore/
      |- __init__.py             # Exposes RankAndScore and analyze_sample
      |- config.py               # Robust, dynamic path configurations
      |- metscore.py             # Singscore ranking and scoring function
      |- analyze_sample.py       # Visium loading and preprocessing pipeline
   |- setup.py                   # Package setup script
|- README.md                     # This file
|- data/                         # Large data storage (git-ignored)
   |- raw/                       # Raw transcriptomics data and metadata
      |- Samples/                # Folder containing Visium samples
      |- patient_metadata.csv    # Patients and samples metadata
      |- met-high-genes-human.csv # Met-high gene signature
      |- met-low-genes-human.csv  # Met-low gene signature
      |- MetHigh-MetLow Genes.xlsx
   |- processed/                 # Intermediate processed data
   |- cleaned/                   # Final output data
|- scripts/                      # Utility scripts
   |- reference_preparation.py   # Script to download & prepare single-cell reference atlas
   |- submit_ref.sh              # SLURM script to run reference preparation
   |- archive/                   # Archive for no longer used scripts
|- environment.yml               # Conda environment definition
|- output/                       # Backwards-compatible outputs directory
```

## Setup & Installation

### 1. Set Up Conda Environment
Create or activate the conda environment using the provided `environment.yml` file:
```bash
conda env create -f environment.yml
# Or if it already exists, activate it:
conda activate /scratch/rprest2/Spatial-MetScore/.conda/Spatial
```

### 2. Install the local `spatial_metscore` package
Install the local custom package in **editable mode** within the active conda environment. This allows you to import functions (e.g. `from spatial_metscore import analyze_sample`) from any notebook or script regardless of your current directory:
```bash
pip install -e spatial_metscore/
```

## Running the Analyses

- **Preprocess reference atlas**: Submit the reference preparation job to the cluster using:
  ```bash
  sbatch scripts/submit_ref.sh
  ```
- **Run notebooks**: Launch Jupyter Notebook or JupyterLab and navigate to `notebooks/` to run step-by-side visualizations, MetScore profiling, and deconvolution analyses.
